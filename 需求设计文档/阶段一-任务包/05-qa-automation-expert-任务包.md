# QA Automation Expert (QA) 任务包 - 阶段二

## 任务概览
负责新模块的单元测试与集成测试。

## 任务清单

### [TASK-QA-2-01] File IO 测试
*   **目标**: 保证文件读写的一致性。
*   **内容**:
    *   构造复杂的 Scene 数据，测试 `encode` -> `decode` 是否无损。
    *   测试异常 JSON 的容错处理。

### [TASK-QA-2-02] History System 测试
*   **目标**: 验证撤销重做逻辑。
*   **内容**:
    *   模拟连续操作（画墙 -> 移动 -> 删除），验证 Undo 后的状态恢复。
    *   测试 Transaction 原子性。

### [TASK-QA-2-03] Algorithm 测试
*   **目标**: 验证几何算法准确性。
*   **内容**:
    *   测试多边形布尔运算的边界情况（自交、共边）。
    *   测试空间索引的查询准确性。

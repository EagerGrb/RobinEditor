- frontend/：前端 UI / 输入焦点 / 交互
- graphics/：图形内核 / 工具链 / 选择与拖拽
- rendering/：渲染引擎 / dirty-rect / 画布输出
- integration/：总线/中介者/模块协作/端到端链路

### 记录字段

- 用例/问题 ID
- 严重程度（Blocker/Critical/Major/Minor）
- 模块（建议 Owner）
- 环境
- 前置条件
- 复现步骤
- 预期结果
- 实际结果
- 怀疑原因（技术判断）
- 影响范围
- 修复建议（可执行）
- 验收标准 / 回归点
